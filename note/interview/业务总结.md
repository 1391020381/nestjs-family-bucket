- 使用Vue.js编译器API：Vue.js提供了一个名为Vue.compile()的编译器API，可以将模板字符串编译为渲染函数

```
import { createApp, compile } from 'vue';

const app = createApp({
  data() {
    return {
      message: 'Hello, Vue.js!'
    };
  },
  render: null
});

const { render } = compile('<div>{{ message }}</div>');
app.render = render;
app.mount('#app');


```
